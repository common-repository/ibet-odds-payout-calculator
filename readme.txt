== Installation ==
= From your WordPress dashboard =
1. Visit "Plugins > Add New"
2. Search for "Odds Payout Calculator Widget"
3. Activate Exchange Rates Widget from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.

= From WordPress.org =
1. Download Odds Payout Calculator Widget.
2. Upload the "Odds Payout Calculator Widget" directory to your "/wp-content/plugins/" directory, using your favorite method (ftp, sftp, scp, etc...).
3. Activate Odds Payout Calculator Widget from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.

= From WordPress "Add Plugins" =
1. Download Odds Payout Calculator Widget.
2. Go to yourdomain.com/wp-admin/plugin-install.php
3. Press button "Upload Plugin"
3. Choise "Odds Payout Calculator Widget" zip archive and press button "Install Now".
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.

== Frequently Asked Questions ==
= How to install a widget on an arbitrary page? =
1. You can generate a shortcode manual
Example:
[odds_payout_calculator lg="ru" tz="0" fm="EUR" to="USD" st="info" bg="FFFFFF" lr="1" rd="0"][/odds_payout_calculator]
Params:
lg="ru" - languages, use: en, ru, it, fr, es, de, cn, pt, ja, id, hi
tz="0" - timezone
fm="EUR" - currency code from currencyrate.today/different-currencies
to="USD" - currency code to currencyrate.today/different-currencies
st="info" - theme (color scheme) primary, info, danger, warning, gray, success (used bootstrap3 color classes)
bg="FFFFFF" - background color of iframe, if you use param: rd = "0"
lr="1" - large widget
rd="0" - 1 - straight corners, 0 - rounded corners
2. You can generate a shortcode automatic
a. Go to yourdomain.com/wp-admin/widgets.php page;
b. Add widget "Odds Payout Calculator";
c. Select options, click save;
d. Copy shortcode from textarea.
3. Languages
en - English
ru - Русский
it - Italiano
fr - Français
es - Español
de - Deutsch
cn - 中国
pt - Português
ja - 日本語
id - Bahasa Indonesia
hi - हिन्दी
